Security (template)

Deze pagina helpt vertrouwen te bouwen bij klanten en auditors. Pas hem aan op je echte hosting.

Basismaatregelen

  • HTTPS (TLS) verplicht
  • Rol-gebaseerde toegang (QC / Directie / Auditor)
  • Immutable auditlog (wie/wat/wanneer)

Opslag

  • Data gescheiden per bedrijf (tenant)
  • Bestanden in object storage (S3/Blob) met toegangscontrole
  • Backups + hersteltest
